# 🤖 MultiMind AI

Multi-Agent AI Debate System - Bikin AI debat & saling koreksi!

## ✨ Features

- 🤖 **Multi-Agent Debate**: DeepSeek + Gemini saling review
- 💰 **Token Efficient**: Compressor + Warning system
- 📎 **File Upload**: PDF, Excel, Word, Images, Code
- 🧠 **Session Memory**: Continue atau Standalone mode
- 👥 **Multi-User**: Database terpisah per user
- 🆓 **Mostly FREE**: Gemini gratis, DeepSeek murah

## 🚀 Quick Deploy

### Streamlit Cloud (GRATIS)
1. Upload ke GitHub
2. Connect ke share.streamlit.io
3. Set secrets (API keys)
4. Deploy!

### HuggingFace Spaces (GRATIS)
1. Upload ke HuggingFace Space
2. Pilih Streamlit SDK
3. Set environment variables
4. Deploy!

1. Login sebagai user
